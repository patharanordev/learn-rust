# Linter

We use `clippy` to collection of lints to catch common mistakes and improve your Rust code.

## Installation

```sh
rustup update
rustup component add clippy
```

## Validate your source code

For more info, please refer to [clippy](https://github.com/rust-lang/rust-clippy).

```sh
cargo clippy
```

or auto-fix it:

```sh
cargo clippy --fix
```
