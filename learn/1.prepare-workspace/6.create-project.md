# Create project

## New project

```sh
cargo new <PROJECT_NAME>
```

### Install dependency

```sh
cargo add <DEPENDENCY_NAME>
```

### Validate project with dependency and syntax

```sh
cargo check
```

### Compile only

```sh
cargo build
```

### Compile & Run project

```sh
# at root of project
cargo run
```

## Add Rust analyzer

### Extension

To validate Rust syntax in IDE (ex. VSCode, Cursor, ...) when you are coding:

- Install [rust-analyzer](https://marketplace.cursorapi.com/items/?itemName=rust-lang.rust-analyzer) to your IDE.
- It works fine when open Rust project and found Cargo.toml in root of project only.
- Set `settings.json` of your IDE to support it:

    ```json
    {
    // ... 
        "rust-analyzer.checkOnSave":true,
        "rust-analyzer.server.extraEnv": { "RUSTUP_TOOLCHAIN": "stable" },
        "rust-analyzer.workspace.discoverConfig": null,
        "rust-analyzer.debug.engineSettings": {},
        "rust-analyzer.procMacro.ignored": {
            "leptos_macro": [
                "server"
            ]
        },
        "rust-analyzer.cargo.features": [
            "ssr"
        ],
    // ... 
    }
    ```

- Enjoy!!!.

### Update `Cargo.toml`

Adding `ssr`:

```toml
# ...
[features]
ssr = []
# ...
```
